Sistema de Admisión ULEAM - Versión Estudiante

Autores: Jean Pierre Flores Piloso, Braddy Londre Vera, Bismark Cevallos
Materia: Programación Orientada a Objetos - 3ro TI "C"
Año: 2025

Descripción:
Este sistema simula el proceso de admisión universitaria de la ULEAM.
Incluye módulos para postulantes, inscripciones, evaluaciones, puntajes y asignación de cupos.

Características:
- Uso de clases, herencia, abstracción y polimorfismo.
- Diseño simple y didáctico, adaptado a nivel estudiante.
- Modo DEMO activado (no se ejecutan procesos reales de evaluación).

Instrucciones:
1. Ejecutar el archivo main.py con Python 3.10+
2. Navegar por el menú y explorar las opciones disponibles.
3. Revisar los modelos dentro de la carpeta 'models' para entender la estructura del sistema.

Hecho con fines educativos y demostrativos para la materia POO ULEAM 2025.
